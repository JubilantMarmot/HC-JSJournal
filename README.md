# JSJournal

Uses localstorage to store the journal entries.
install with `bun install` and start with `bun start`.

This was so I could learn localstorage.

![](./1.png)
![](./2.png)

## Deployment

1. Clone the repo
2. Run `bun install`
3. Run `bun backend/server.js` and it will start on port 3000
